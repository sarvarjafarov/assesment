{# ── Jobs & Positions ── #}

Jobs

Create positions and organize them into campaigns

Positions

Each position represents a role you're hiring for. When creating one you define:

Title & description — role name and detailed job description
Department & location — for organizing and filtering
Employment type — full-time, part-time, contract, internship
Assessment type — which assessment candidates should complete
Published status — toggle visibility on your public vacancy page
{% if can_use_ai_hiring %}
AI Screening Pro — automatic resume scoring and candidate automation
{% endif %}
Campaigns

Group related positions into a hiring project (e.g., "Q1 Engineering Hiring") to track them together and share progress with your team.

Candidate List

{# ── Assessments ── #}

Assessments

AI-scored assessments for every role and skill level

Browse the library — all assessment types with descriptions and estimated time
Manage sessions — view invited candidates, progress, and scores
Invite candidates — send assessment links via email
View results — detailed scores, AI feedback, and session data
Export data — download results as CSV
Tip: Attach an assessment to a position — candidates who apply via your vacancy page receive it automatically.
{% if can_use_ai_hiring %} {# ── AI Screening ── #}

AI Screening Pro

Automate the first stage of your hiring pipeline

1

Enable on a Position

Edit any position and toggle "Enable AI Screening". Choose your automation level:

Screen only Screen + auto-send Full automation
2

Candidates Get Scored

When candidates apply, their resume is analyzed and scored 0–100. Candidates scoring 60+ are marked as "passed" and move forward automatically.

3

Review Results

The Candidates page shows an "AI Score" column. On each candidate's detail page, submit a human review decision and view the full AI Processing Log.
